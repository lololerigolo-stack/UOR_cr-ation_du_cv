# UOR - Création d'un CV 

Projet réalisé dans le cadre du module **Informatique Fondamentale** 
de la Licence 1 Informatique à l'IED Paris 8.

## Description

Site web personnel multi-pages présentant un CV et un portfolio sur la 
thématique du sport (vélo, musculation, natation, sport de combat).

## Technologies utilisées

- **HTML5** - structure des pages
- **CSS3** - mise en forme 
- **PHP** - traitement des formulaires
- **MySQL** - stockage des données 
- **MongoDB** - base NoSQL 
- **JavaScript** 

## Pages du site

- `Mon_cv.html` - CV principal
- `velo.html` - page Vélo
- `musculation.html` - page Musculation
- `natation.html` - page Natation
- `sport_combat.html` - page Sport de combat
- `Mon_formulaire.php` - formulaire 
- `javascript.html` - page interactive , chrono, titre interactif
